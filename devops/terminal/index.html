    httpie

    httpie est un client en ligne de commande pour tester les API en http et https. Il possède une syntaxe simple et élégante, et se combine très bien avec l'outil suivant pour traiter les données JSON.

    Suivant l'adresse sur laquelle on fait la requête, httpie détecte automatiquement si la requête demandée sera une requête POST ou GET.

    Exemple

    Une requête GET sur mon API perso fastapi.mathieuklimczak.com/inferences/ se fait alors simplement comme ça.

    jq

    jq est une outil de parsing des données json dans le terminal. Il est capable de slicer, filtrer et transformer la données structurées de la même façon que sed, awk ou grep.

    Exemple

    Pour ne récupérer que les données, sans le header de la requête, on peut alors combiner httpie et jq.
