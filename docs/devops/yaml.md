# Just enough YAML

Principalement utilisé pour écrire des fichiers de configuration.

Paire clé/valeur

```yaml
Fruit: Apple
Vegetable: Carrot
Liquid: Water
Meat: Chicken
```

Array/Lists

```yaml
Fruits:
- Apple
- Orange
- Banana

Vegetables:
- Carrot
- Cauliflower
- Tomato
```

Dictionary/Map

```yaml
Banana:
  Calories: 105
  Fat: 0.4 g
  Carbs: 27 g

Grapes:
  Calories: 62
  Fat: 0.3 g
  Carbs: 16 g
```

Indentation dans YAML.


Clé/valeur/Dictionnaire/Liste
```yaml
Fruits:
  - Banana:
      Calories: 105
      Fat: 0.4 g
      Carbs: 27 g

  - Grapes:
      Calories: 62
      Fat: 0.3 g
      Carbs: 16 g
```
